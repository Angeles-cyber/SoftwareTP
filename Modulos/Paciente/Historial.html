Historial de Vacunación

¿Cómo Realizar Cambios o Actualizaciones?

En esta sección puedes visualizar el historial de vacunación de un paciente, incluyendo las vacunas que se ha puesto, las que le faltan y las que están en proceso. A continuación, te explicamos cómo puedes realizar modificaciones o agregar nuevas citas sin perder la trazabilidad.

Agregar o Modificar Vacunas

Para actualizar el historial de vacunación, como agregar nuevas vacunas administradas o cambios en la programación de citas, debes acceder a la base de datos correspondiente. Recuerda que siempre se debe registrar una vacuna bajo el nombre de la persona y el centro de salud donde se administró. Si no existe una cita programada, puedes añadir una nueva para cada vacuna pendiente.

Actualizar Citas en Proceso

Para modificar una cita en proceso, ya sea para cambiar la fecha o el centro de salud, puedes hacerlo directamente desde la tabla de "Vacunas en Proceso". Si se requiere cancelar una cita, haz clic en el botón "Cancelar". Los cambios se reflejarán automáticamente en la base de datos del sistema para mantener un registro exacto y actualizado.
